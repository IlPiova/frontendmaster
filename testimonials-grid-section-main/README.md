### What I learned

I’ve never had any trouble using the `grid-template-areas` property; it’s one of my favourite tools because it’s very creative and easy to organise and manage. I’m pleased to have developed this project in less time than previous ones and, above all, **to have avoided repeating the usual mistakes** with the sizing of the body and images.

### Continued development

This encourages me to keep studying `Grid` and `Flexbox`. I also need to get better at creating box shadows, which often look awful when I first set them up.
