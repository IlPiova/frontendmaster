### What I learned

I’ve learnt to make better use of `CSS Grid` and, above all, to style images more effectively.

### Continued development

I had a lot of trouble with the sizing of the cells and, consequently, the entire grid. In the next iteration of this project, I’ll try to organise the dimensions of the various components more effectively, perhaps by making use of CSS properties such as `calc()`, `min()` and `clamp()`.
